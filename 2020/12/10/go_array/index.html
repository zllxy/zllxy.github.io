Go语言中的Array与Slice

发表于2020-12-10|更新于2020-12-10|GoLang学习笔记
|字数总计:819|阅读时长:3分钟|阅读量:

array

  • 定义方式：

    var arr [n]type //n表示数组的长度，type表示存储元素的类型

  • 对数组的操作和其他语言类似，都是通过[]来进行读取或赋值。例如：

    arr := [5]int{1,2,3,4,5}
fmt.Println(arr[0])

  • 数组是一种复合类型，长度是数组类型的一部分。因此，[3]int 和 [2]int 是两种不同的类型。

  • 数组的长度不能更改

  • 数组之间的赋值是值的赋值。当把一个数组作为参数传入函数的时候，会把这个数组再复制一份传入，而不是它的指针，如果需要用到指针，需要用到slice。

    arr1 := [3]int{1, 2, 3}
arr2 := arr1
fmt.Printf("%p", &arr1) //0xc00000c380
fmt.Printf("%p", &arr2) //0xc00000c3a0

slice

  • 定义方式：

    var slice []type //与数组的定义方式类似，至少缺少了长度

  • 创建数组切片（slice）：

    基于数组，通过 array[i:j] 来获取，其中 i 是数组的开始位置，j 是结束位置，但不包含array[j]，它的长度是 j-i :

    arr3 := [5]int{1, 2, 3, 4, 5}
slice := arr3[:]

    使用 make() 函数，第一个参数表示类型，第二个参数表示创建元素的个数，第三个参数表示数组切片的容量：

    slice1 := make([]int, 5, 10)

    直接创建：

    slice2 := []int{1, 2, 3}

  • slice是一种引用类型，由三个部分构成：指针、长度和容量。指针指向数组起始下标，长度对应切片中元素的个数，容量则是切片起始位置到底层数组结尾的位置

数组切片相关内置函数

  • len() 获取slice的长度

  • cap() 获取slice的最大容量

    var slice3 = make([]int, 5, 10)

fmt.Println("len(slice3):", len(slice3))
fmt.Println("cap(slice3):", cap(slice3))

  • append() 向slice中追加元素，返回一个与slice类型相同的slice。使用append会改变slice 所引用的数组的内容。如果slice没有剩余空间的时候，会动态分配一个新的数组空间，而原有的数组不会发生变化。

    slice := make([]int, 5, 6)
slice1 := append(slice, 1) //未超出slice容量
slice2 := append(slice, 1, 2) //超出slice的容量

fmt.Println(slice) //[0 0 0 0 0]
fmt.Println(slice1) //[0 0 0 0 0 1]
fmt.Println(slice2) //[0 0 0 0 0 1 2]

fmt.Printf("%p", slice) //0xc000010450
fmt.Printf("%p", slice1) //0xc00001045，地址与slice相同
fmt.Printf("%p", slice2) //0xc000054060，地址与slice不同

//改变slice的值
slice[1] = 1

fmt.Println(slice) //[0 1 0 0 0]
fmt.Println(slice1) //[0 1 0 0 0 1] //发生变化
fmt.Println(slice2) //[0 0 0 0 0 1 2] //为发生变化

  • copy() 将元素从一个数组切片复制到另外一个数组切片，返回复制元素的个数。如果两个切片长度不一致，按其中较小的那个数组切片的元素个数进行复制

    slice1 := []int{1, 2, 3, 4, 5}
slice2 := []int{6, 7, 8, 9, 10}
copy(slice1, slice2) //将slice2的元素复制到slice1中
fmt.Println(slice1) //[6 7 8 9 10]
fmt.Println(slice2) //[6 7 8 9 10]



//将slice3中的元素复制到slice4中，由于slice4的长度比slice3的小，
//所以只会复制slice3中的前3个元素复制到slice4中

slice3 := []int{1, 2, 3, 4}
slice4 := []int{5, 6, 7}
copy(slice4, slice3)
fmt.Println(slice3) //[1 2 3 4]
fmt.Println(slice4) //[1 2 3]
